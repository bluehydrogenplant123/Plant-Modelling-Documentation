# Chemware Engineering Capstone

Our project is a user interface and model version management tool for modeling and simulating industrial plants. The user interface for a model is represented by a network of nodes and edges that the user can freely modify and edit. This GUI is intended to replace the current method used for creating factory diagrams which is Excel tables. These excel tables are limited in terms of readability and do not provide an appropriate presentation of the network topology of a factory plant. Our GUI software fixes this by still providing the user data driven features of excel but also the readability and topology of a GUI software.

This project is the capstone for Team 18: Chemware Engineering for the course SFWRENG 4G06 at McMaster University.
