# Interface Overview

Use this page when you first open the HyProNet modeling workspace and need to understand what each part of the screen is for.

![HyProNet model workspace with module tabs, model toolbar, palette, canvas, save controls, zoom controls, and minimap](./images/model-interface-overview.png)

## Main Screen Areas

| Area | What You See | What It Is For |
| --- | --- | --- |
| Module tabs | **Model**, **Materials**, **Economic**, **Calc Type**, **Analysis**, **Set Run**, **Run**, **Run Result**, **Multi-TP**, **TP Analysis**, **System**, **Display**, and **Help** | Switch between major working modes. The active tab controls which toolbar actions are shown below it. |
| Model toolbar | **New Model**, **Open**, **Duplicate Network**, **Extract Selection**, **Subnetwork Editor**, **Import Subnetworks**, **Equation Writing**, **+Constr**, **SoluAlgoLib**, **Verify Model**, **Base TP**, **Specs**, **Import**, and **Export** | Start, open, duplicate, verify, import, export, or prepare the current model. |
| Palette | The left panel labeled **Palette**, with unit types such as **ACC**, **AFT_BURN**, **ASU**, **ATR**, **BLOWER**, **COMBUST**, **COMPR**, and **DIVIDER** | Find the plant units that can be added to the model canvas. |
| Canvas | The large dotted workspace in the center | Build and inspect the process network. In the screenshot, the canvas is empty because no nodes have been placed yet. |
| Save and model state | **Save**, **Refresh Off**, the network name field such as **D1**, **Back**, and the last-saved message | Save the current network, check auto-save status, rename or identify the current diagram, and return to the previous page. |
| Floating control | The dark floating bar near the upper-right side of the canvas | Shows a compact canvas control while working in the modeling area. |
| Zoom controls | The controls at the bottom-left of the canvas | Zoom in, zoom out, reset the view, or use the available canvas view control. |
| Minimap | The small overview panel at the bottom-right | Check where the visible canvas area sits inside the larger workspace. |

## How To Orient Yourself

1. Start from the top module tabs and confirm the correct working mode is selected.
2. Use the second toolbar row for actions that belong to the selected mode.
3. Use the **Palette** when you need a plant unit for the model.
4. Work in the dotted canvas area when building or inspecting the network.
5. Watch the save status in the top-right corner after clicking **Save** or after auto-save runs.
6. Use the zoom controls and minimap when the diagram becomes larger than the visible canvas.

## Result

You can identify the main workspace regions before following the task-based tutorials for model creation, node editing, equations, economic data, plant measurements, time periods, and computation runs.
