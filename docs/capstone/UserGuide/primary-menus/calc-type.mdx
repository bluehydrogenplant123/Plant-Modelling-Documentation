# Calc Type Menu

The **Calc Type** primary menu sets the calculation mode for the current network. The selected mode affects node variables and specification behavior.

## Secondary Buttons

| Button | What It Is For |
| --- | --- |
| **Simulation** | Use simulation mode. |
| **Optimization** | Use optimization mode. |
| **DataRec** | Use data reconciliation mode. |
| **ParamUpdt** | Use parameter update mode. |

When you choose a different calculation type, the application asks you to confirm the switch.
