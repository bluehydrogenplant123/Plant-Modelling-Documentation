# Calc Type Menu

The **Calc Type** primary menu sets the calculation mode for the current network. The selected mode affects which node variable specifications are used in **TP Specs**.

## Secondary Buttons

Click a button name to open the detailed page for that calculation type.

| Button | What It Is For |
| --- | --- |
| **[Simulation](./calc-type/simulation)** | Use simulation mode. |
| **[Optimization](./calc-type/optimization)** | Use optimization mode. |
| **[DataRec](./calc-type/data-rec)** | Use data reconciliation mode. |
| **[ParamUpdt](./calc-type/param-updt)** | Use parameter update mode. |

Clicking the selected type opens **TP Specs**. Clicking a different type opens **Switch Calc Type** first, then changes the mode after confirmation.

Calc Type buttons can be disabled while computation-related read-only rules are active.
