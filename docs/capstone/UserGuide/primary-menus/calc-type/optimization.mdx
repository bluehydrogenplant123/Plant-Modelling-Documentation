# Optimization

Use **Optimization** when the current network should run in optimization mode.

## Where To Find It

1. Select the **Calc Type** primary menu.
2. Click **Optimization** in the secondary button row.

## What It Opens/Does

If **Optimization** is already selected, clicking it opens **TP Specs**. If another calculation type is selected, clicking **Optimization** opens **Switch Calc Type**. Confirming the switch changes the calculation mode and opens **TP Specs**.

## Basic Steps

1. Open **Calc Type**.
2. Click **Optimization**.
3. If **Switch Calc Type** appears, review the source and target modes.
4. Click **Confirm** to switch to **Optimization**.
5. Review the rows in **TP Specs**.
6. Edit values, specs, or bounds as needed.
7. Click **Save**.

## Result

The network uses **Optimization** as its calculation mode, and the saved **TP Specs** rows reflect that mode.
