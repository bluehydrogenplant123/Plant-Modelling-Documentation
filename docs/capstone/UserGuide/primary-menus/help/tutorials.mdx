# Tutorials

Use **Tutorials** as a reference for the current in-app tutorial placeholder.

## Where To Find It

1. Open HYPRONET.
2. Select the **Help** primary menu.
3. Find **Tutorials** in the secondary button row.

## What It Does

**Tutorials** is disabled in the current header implementation. It is reserved for future in-app tutorial access.

Use the online User Manual when this in-app placeholder is unavailable.

## Basic Steps

1. Select the **Help** primary menu.
2. Confirm **Tutorials** is disabled.
3. Open the online User Manual instead.
4. Choose the page that matches the workflow you need to complete.

## Result

No in-app tutorial page opens from this button in the current implementation.
