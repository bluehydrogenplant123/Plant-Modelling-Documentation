# Display Button

The **Display** button is a primary header action for filtering what is visible on the canvas.

## What It Does

| Option | What It Shows |
| --- | --- |
| **All nodes** | Shows every node on the canvas. |
| **Normal nodes** | Shows regular nodes and hides subnetwork wrapper nodes. |
| **Subnetwork nodes** | Shows subnetwork wrapper nodes. |

Click **Display**, choose the node visibility option, and click **Apply**.
