# Model Menu

The **Model** primary menu is selected in the annotated screenshot. It controls the main model lifecycle: creating, opening, duplicating, extracting, verifying, importing, and exporting the network.

## Secondary Buttons

| Button | What It Is For |
| --- | --- |
| **New Model** | Start a new model diagram. |
| **Open** | Open an existing network. |
| **Duplicate Network** | Save a copy of the current network. |
| **Extract Selection** | Extract selected canvas elements into a separate selection workflow. |
| **Subnetwork Editor** | Save or edit a subnetwork from the current model context. |
| **Import Subnetworks** | Import subnetwork definitions into the current network. |
| **Equation Writing** | Open the equation editor for equations and constraints. |
| **+Constr** | Open the constraint module. |
| **SoluAlgoLib** | Open the solution algorithm library editor. |
| **Verify Model** | Validate the model before advanced workflows or computation. |
| **Base TP** | Edit the base time-period duration. |
| **Specs** | Open base specification settings. |
| **Import** | Import diagram data. |
| **Export** | Export diagram data. |
