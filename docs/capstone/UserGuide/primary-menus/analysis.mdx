# Analysis Menu

The **Analysis** primary menu opens model analysis workflows from the secondary button row.

## Secondary Buttons

Click a linked button name to open the detailed page for that workflow. Disabled placeholders are listed for orientation only.

| Button | What It Is For |
| --- | --- |
| **[Plant Measurements](./analysis/plant-measurements)** | Open the plant measurement workflow. |
| **Expenses** | Disabled placeholder for expense analysis. |
| **Objective Function** | Disabled placeholder for objective function analysis. |
| **Revenues** | Disabled placeholder for revenue analysis. |
