# Run Result Button

The **Run Result** button is a primary header action for reviewing computation history.

## What It Does

| Button | What It Is For |
| --- | --- |
| **Run Result** | Open the run result history modal for stored computation results. |

Use this button after a computation has completed and the backend has stored the result records.
