# Dashboard

Use **Dashboard** when you need to create, rename, delete, or add panels to TP Analysis dashboards.

## Where To Find It

1. Open an existing diagram.
2. Select the **TP Analysis** primary menu.
3. Click **Dashboard** in the secondary button row.

## What It Opens

The **Dashboard** manager modal opens. It shows the dashboard list, **New dashboard**, **Rename**, **Delete**, and **Add panel** options.

The available panel types are **Scatter Plot**, **Line Plot**, **Bar Plot**, **Pie Chart**, and **Table**.

## Basic Steps

1. Open **Dashboard**.
2. Select a dashboard from the list, or enter a name under **New dashboard** and create one.
3. Use **Rename** or **Delete** when you need to manage the selected dashboard.
4. Choose an **Add panel** type.
5. Configure the enabled panel fields.
6. Click **Add panel**.
7. Close the modal when finished.

## Result

The dashboard list and selected dashboard panels are updated. New panels become available to view from **Visualization**.
