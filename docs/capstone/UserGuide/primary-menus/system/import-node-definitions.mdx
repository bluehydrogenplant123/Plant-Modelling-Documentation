# Import Node Definitions

Use **Import Node Definitions** as a reference for the current system-node-definition import placeholder.

## Where To Find It

1. Open HYPRONET.
2. Select the **System** primary menu.
3. Find **Import Node Definitions** in the secondary button row.

## What It Does

**Import Node Definitions** is currently an inline **System** button. It may be disabled by computation-related rules, and its import handler is a placeholder in the current UI.

It does not currently open a complete node-definition import workflow.

## Basic Steps

1. Select the **System** primary menu.
2. Check whether **Import Node Definitions** is enabled.
3. If it is disabled, continue using the existing node definitions.
4. If it appears enabled, treat it as a placeholder unless a future version adds an import dialog.

## Result

No system node-definition import flow is completed from this button in the current implementation.
