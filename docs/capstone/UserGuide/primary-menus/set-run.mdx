# Set Run Menu

The **Set Run** primary menu is used before starting computation. It lets you choose and configure the solver and algorithm options loaded for the project.

## Secondary Buttons

| Button | What It Is For |
| --- | --- |
| **Solver** | Open the solver dropdown and choose a solver configuration. |
| **Algorithm** | Open the algorithm dropdown and choose an algorithm configuration. |

Choosing a solver or algorithm opens the shared run configuration modal.
