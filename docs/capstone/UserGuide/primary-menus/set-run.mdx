# Set Run Menu

The **Set Run** primary menu is used before computation. It lets you open solver and algorithm configuration panels loaded from the current project data.

## Secondary Buttons

Click a button name to open the detailed page for that configuration workflow.

| Button | What It Is For |
| --- | --- |
| **[Solver](./set-run/solver-configuration)** | Open the solver dropdown and choose a solver configuration. |
| **[Algorithm](./set-run/algorithm-configuration)** | Open the algorithm dropdown and choose an algorithm configuration. |

The dropdown item labels come from the available run configuration names. Choosing a solver or algorithm opens the shared run configuration modal.
