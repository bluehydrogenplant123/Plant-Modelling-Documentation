# Multi-TP Menu

The **Multi-TP** primary menu is used for multi-time-period workflows after the model is ready for time-period-specific configuration.

## Secondary Buttons

Click a linked button name to open the detailed page for that secondary button.

| Button | What It Is For |
| --- | --- |
| **Time Horizon** | Placeholder for time horizon controls. It is disabled in the current header implementation. |
| **View Time Periods** | Placeholder for viewing time periods. It is disabled in the current header implementation. |
| **[TP Node - Model Version Control](./multi-tp/tp-node-model-version-control)** | Open the TP node model-version control workflow after verification. |
| **[Global TP](./multi-tp/global-tp)** | Configure global time-period ranges and duration settings. |
| **[TP Specs](./multi-tp/tp-specs)** | Review or edit TP-specific specification values in Multi-TP mode. |
| **[Economic](./multi-tp/economic)** | Open Multi-TP economic panels when Multi-TP ranges are available. |
