# Economic Menu

The **Economic** primary menu opens economic data panels for the current diagram. Use it when the model needs cost, revenue, demand, or supply data before analysis or computation.

## Secondary Buttons

Click a button name to open the detailed page for that secondary button.

| Button | What It Is For |
| --- | --- |
| **[Cost & revenue](./economic/cost-revenue)** | Edit cost and revenue rows, including entity and mapping data. |
| **[Demand & Supply](./economic/demand-supply)** | Edit demand and supply rows used by economic workflows. |

## When To Use It

Use **Economic** after the model structure exists and before running workflows that depend on economic rows.

Economic panels can become read-only while computation-related rules are active.
