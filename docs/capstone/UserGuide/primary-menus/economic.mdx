# Economic Menu

The **Economic** primary menu opens the economic data controls for the current diagram. Use it when the model needs cost, expense, revenue, objective, entity, or mapping data before analysis or computation.

## Secondary Area

| Control Area | What It Is For |
| --- | --- |
| **Economic controls** | Open economic panels, edit economic rows, import economic workbook data, export rows, and save economic data for the diagram. |

## When To Use It

Use **Economic** after the model structure exists and before running workflows that depend on cost or revenue data.
