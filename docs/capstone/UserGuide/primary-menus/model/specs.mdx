# Specs

Use **Specs** when you need to edit base specification values for the model.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **Specs** in the secondary button row.

## What It Opens

The **Specs** button opens the shared specification panel in base mode. It loads diagram node data and lets you edit specification rows for the base time period.

## Basic Steps

1. Open **Specs**.
2. Review the loaded specification rows.
3. Edit the required base specification values.
4. Click **Save**.

## Result

Base specification changes are saved for the current diagram.
