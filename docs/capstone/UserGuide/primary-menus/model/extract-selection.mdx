# Extract Selection

Use **Extract Selection** when you need to turn selected canvas content into a separate network.

## Where To Find It

1. Select nodes or streams on the canvas.
2. Select the **Model** primary menu.
3. Click **Extract Selection** in the secondary button row.

## What It Opens

The **Extract Selection** dialog contains **Network Name**, **Description**, selected node and stream counts, and a **Preview** panel. The preview lets you inspect and adjust the selected content before creating the extracted network.

## Basic Steps

1. Review the selected node and stream counts.
2. Enter a **Network Name**.
3. Enter a **Description** if needed.
4. Review the **Preview**.
5. Click **Create**.

## Result

The selected canvas elements are saved as a separate extracted network.
