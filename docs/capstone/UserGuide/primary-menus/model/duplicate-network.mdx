# Duplicate Network

Use **Duplicate Network** when you need a saved copy of the current network under a new name.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **Duplicate Network** in the secondary button row.

## What It Opens

The **Duplicate Network** dialog asks for a **Network Name**. It checks whether the name is empty, already used, or available.

## Basic Steps

1. Enter a new **Network Name**.
2. Wait for the name availability message.
3. Click **Duplicate**.
4. Click **Cancel** if you do not want to create the copy.

## Result

The current network is duplicated and saved with the new name.
