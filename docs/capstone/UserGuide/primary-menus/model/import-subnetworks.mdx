# Import Subnetworks

Use **Import Subnetworks** when you need to add existing subnetwork definitions to the current network.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **Import Subnetworks** in the secondary button row.

## What It Opens

The **Import Subnetworks** dialog lists available subnetwork blueprints. Each row shows a checkbox, icon preview, model name, and port count.

## Basic Steps

1. Review the available subnetworks.
2. Select one or more checkboxes.
3. Click **Import Selected**.
4. Click **Cancel** if you do not want to import anything.

## Result

The selected subnetwork definitions are imported into the current network context.
