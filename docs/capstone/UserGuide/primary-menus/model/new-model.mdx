# New Model

Use **New Model** when you need to start a new model diagram.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **New Model** in the secondary button row.

## What It Opens

The **New Model** dialog contains **Diagram Name**, **Select Domain**, **Calculation Type**, and **Description** fields. It also includes **Create Diagram**, **Import Diagram**, and **Logout** actions.

## Basic Steps

1. Enter a diagram name.
2. Select the domain for the new model.
3. Choose the calculation type.
4. Enter a short description.
5. Click **Create Diagram**.

## Result

A new diagram is created and opened as the current modeling workspace.
