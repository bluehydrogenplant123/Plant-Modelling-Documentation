# Verify Model

Use **Verify Model** when you need to mark the current diagram as verified before advanced workflows or computation.

## Where To Find It

1. Open an existing diagram.
2. Select the **Model** primary menu.
3. Click **Verify Model** in the secondary button row.

## What It Does

The button sends a verify request for the current diagram. After verification succeeds, the button label changes to **Verified** and the diagram is treated as verified.

## Basic Steps

1. Confirm the network is ready to verify.
2. Click **Verify Model**.
3. Wait for the verification status message.

## Result

The current diagram is marked as verified. If it is already verified, the UI warns that the diagram is already verified.
