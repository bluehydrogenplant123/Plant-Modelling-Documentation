# Base TP

Use **Base TP** when you need to edit the base time-period duration for the current network.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **Base TP** in the secondary button row.

## What It Opens

The **Base TP** dialog contains **Duration** and **Duration Unit**. The dialog notes that the Base TP duration is synced to the network and its subnetworks.

## Basic Steps

1. Enter the base **Duration**.
2. Choose the **Duration Unit**.
3. Click **Save**.
4. Click **Close** when finished.

## Result

The diagram-level Base TP duration is saved. The button is disabled after entering Multi-TP mode.
