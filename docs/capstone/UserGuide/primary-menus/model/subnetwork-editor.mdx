# Subnetwork Editor

Use **Subnetwork Editor** when you need to define or edit a reusable subnetwork from the current model context.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **Subnetwork Editor** in the secondary button row.

## What It Opens

The **Subnetwork Editor** dialog includes fields for **Name** and **Icon SVG**, an icon **Preview**, **Upload Icon**, and a **Port Mappings** section. Port mappings define how internal nodes and ports are exposed on the subnetwork.

## Basic Steps

1. Enter a subnetwork **Name**.
2. Provide SVG icon code or click **Upload Icon**.
3. Click **Add Port Mapping** for each port that should be exposed.
4. Choose the node, source port, location, and port name for each mapping.
5. Save the subnetwork from the editor.

## Result

The subnetwork definition is saved and can be reused or imported into another network.
