# Development Plan

The folders and files for this folder are as follows:
Makefile: to build the pdf using the tex file.
tex file: Contains all the details of the development plan.
pdf file: Same contents as the tex file just in a more human readble format.

The Development Plan is intended to cover the overall development plan for the User Interface and Model Version Management for Models of Industrial Plants project. The document will cover all necessary information for the development process of the project including team information, development information, information related to IP as well as other related subjects.
