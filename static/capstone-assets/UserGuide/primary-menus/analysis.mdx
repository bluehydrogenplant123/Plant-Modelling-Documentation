# Analysis Menu

The **Analysis** primary menu opens analysis workflows for the current model.

## Secondary Buttons

| Button | What It Is For |
| --- | --- |
| **Plant Measurements** | Open the plant measurement workflow. |
| **Expenses** | Placeholder button for expense analysis. It is disabled in the current header implementation. |
| **Objective Function** | Placeholder button for objective function analysis. It is disabled in the current header implementation. |
| **Revenues** | Placeholder button for revenue analysis. It is disabled in the current header implementation. |
