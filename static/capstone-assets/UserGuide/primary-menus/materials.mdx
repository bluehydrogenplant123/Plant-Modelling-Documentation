# Materials Menu

The **Materials** primary menu opens material data tools used by the model. In the current header implementation, selecting **Materials** shows the material editor entry in the secondary menu row.

## Secondary Buttons

| Button | What It Is For |
| --- | --- |
| **Material Editor** | Open the material editor modal for viewing or editing material data. |

## When To Use It

Use **Materials** before building or running a model when the available material definitions need to be checked or updated.
