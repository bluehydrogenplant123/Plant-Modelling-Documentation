# Materials Menu

The **Materials** primary menu opens material stream tools used by the current model. Selecting **Materials** shows one secondary button.

## Secondary Buttons

Click the button name to open the detailed page for that secondary button.

| Button | What It Is For |
| --- | --- |
| **[Material Editor](./materials/material-editor)** | Open the material editor overlay for stream rows, import, and export. |

## When To Use It

Use **Materials** before building or running a model when stream definitions need to be checked, imported, exported, or cleared.

When the diagram is read-only during computation, the editor can still open, but editing, deleting, and importing are restricted.
