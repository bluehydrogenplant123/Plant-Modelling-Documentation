# Visualization

Use **Visualization** when you need to inspect TP Analysis results through a Metabase dashboard inside HYPRONET.

## Where To Find It

1. Open an existing diagram with TP Analysis results.
2. Select the **TP Analysis** primary menu.
3. Click **Visualization** in the secondary button row.

## What It Opens

The **Visualization** modal opens. It contains **Dashboard name** and **Run name** selectors, an embedded Metabase dashboard, and dashboard zoom controls.

The modal may require `VITE_METABASE_API_KEY`. If the key is missing, the modal shows a Metabase API key message instead of the dashboard.

## Basic Steps

1. Open **Visualization**.
2. Select a **Dashboard name**.
3. Select a **Run name**.
4. Use the dashboard view and zoom controls to inspect the result panels.
5. Close the modal when finished.

## Result

The selected Metabase dashboard is displayed for the selected run. If dashboards, run names, or the Metabase API key are missing, the modal explains what is unavailable.
