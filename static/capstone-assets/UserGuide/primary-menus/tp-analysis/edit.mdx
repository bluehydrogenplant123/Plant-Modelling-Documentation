# Edit

Use **Edit** when you need to open the external dashboard editing interface.

## Where To Find It

1. Open HYPRONET in the browser.
2. Select the **TP Analysis** primary menu.
3. Click **Edit** in the secondary button row.

## What It Opens

The button opens `http://localhost:3001/` in a new browser tab.

This page is outside the HYPRONET main workspace. It depends on the local dashboard editing service being available at that address.

## Basic Steps

1. Click **Edit**.
2. Switch to the new browser tab.
3. Confirm the local dashboard editing page loads.
4. Edit dashboards in the external page as needed.
5. Return to HYPRONET when finished.

## Result

The external dashboard editing page opens in a new tab. If the local service is not running, the browser may show that `http://localhost:3001/` is unavailable.
