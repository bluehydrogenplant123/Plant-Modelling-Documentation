# Run History

Use **Run History** when you need to review previous computation runs for the current diagram.

## Where To Find It

1. Open a diagram.
2. Click the **Run Result** primary header button.

## What It Opens/Does

**Run Result** opens the **Run Result** modal. The modal lists computation history with **Run Name**, **Status**, **Run Time**, and **Action** columns. Each row can include a **Delete** action.

If there is no active diagram ID, the **Run Result** button can be disabled. If no history exists, the modal shows that there is no run history yet.

## Basic Steps

1. Click **Run Result**.
2. Review the **Run Name**, **Status**, and **Run Time** values.
3. Click **Delete** on a row if you need to remove that computation result.
4. Confirm the delete prompt.
5. Close the modal when you finish reviewing the history.

## Result

The modal shows the stored computation history for the current diagram. Deleted result rows are removed after confirmation.
