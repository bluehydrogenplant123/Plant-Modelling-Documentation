# Algorithm Configuration

Use **Algorithm** when you need to review or edit an algorithm configuration before starting computation.

## Where To Find It

1. Select the **Set Run** primary menu.
2. Click **Algorithm** in the secondary button row.
3. Select an algorithm configuration from the dropdown.

## What It Opens/Does

The **Algorithm** dropdown lists run configuration keys that include `Algorithm`. The labels depend on the configurations loaded for the current project.

Selecting an item opens the shared **RunConfigModal** with a title based on the selected type, such as **Composite Algorithm Configuration**. The modal shows configuration attributes in editable controls when editing is allowed. During protected run states, the same panel can be read-only.

## Basic Steps

1. Open the **Algorithm** dropdown.
2. Choose the algorithm configuration you want to inspect.
3. Review the attribute values in the configuration modal.
4. Change editable values if the modal is not read-only.
5. Click **Save** if you made changes, or close the modal if you only reviewed the settings.

## Result

The selected algorithm configuration is ready for use when you open the **Run** computation panel.
