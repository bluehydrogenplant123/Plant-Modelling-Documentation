# Export

Use **Export** when you need to download diagram data from the current network.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **Export** in the secondary button row.

## What It Opens

The **Export Data** dialog lets you choose **Full Data**, **Solver Data**, or **Algorithm Data**. For **Full Data**, you can enter an **Export File Name**. The app adds `.json` automatically when needed.

## Basic Steps

1. Open **Export**.
2. Choose the export mode.
3. Enter an export file name if you selected **Full Data**.
4. Click **Export**.
5. Click **Cancel** if you do not want to export.

## Result

The selected export data is downloaded from the current diagram.
