# SoluAlgoLib

Use **SoluAlgoLib** when you need to configure solution algorithm library rows for the current diagram.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **SoluAlgoLib** in the secondary button row.

## What It Opens

The **SoluAlgoLib** dialog loads diagram-level collections and sets. It lets you edit solution-algorithm library data and save it back to the current diagram.

## Basic Steps

1. Open **SoluAlgoLib**.
2. Review the loaded library rows.
3. Edit the required rows or assignments.
4. Click **Save**.

## Result

The solution algorithm library configuration is saved with the current diagram.
