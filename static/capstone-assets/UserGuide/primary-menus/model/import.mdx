# Import

Use **Import** when you need to import diagram data from a JSON file.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **Import** in the secondary button row.

## What It Opens

The button opens the operating system file picker. It accepts `.json` files and imports the selected diagram data.

## Basic Steps

1. Click **Import**.
2. Choose the JSON file from your computer.
3. Confirm the file selection.
4. Wait for the import to finish.

## Result

The selected diagram data is imported into the application.
