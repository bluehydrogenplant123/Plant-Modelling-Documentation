# Open

Use **Open** when you need to load an existing network.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **Open** in the secondary button row.

## What It Opens

The **Open Network** dialog groups saved networks into **Verified Networks** and **Unverified Networks**. Each listed network can be loaded. Unverified networks also show a **Delete** action.

## Basic Steps

1. Find the network in **Verified Networks** or **Unverified Networks**.
2. Click **Load** for the network you want to open.
3. Use **Export** from the row if you need to download that saved network instead of opening it.
4. Click **Close** if you do not want to open anything.

## Result

The selected network is loaded into the modeling workspace.
