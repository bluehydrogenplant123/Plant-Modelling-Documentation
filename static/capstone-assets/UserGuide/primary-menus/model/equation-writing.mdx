# Equation Writing

Use **Equation Writing** when you need to create or edit diagram equations and constraints from model variables.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **Equation Writing** in the secondary button row.

## What It Opens

The **Equation Writing** dialog provides the equation editor used to build equations from diagram variables, imported variable rows, operators, and time-period suffixes.

## Basic Steps

1. Open **Equation Writing**.
2. Add or select the equation you want to edit.
3. Build the equation from available variables and operators.
4. Save the equation changes.

## Result

The equation is stored with the current diagram and can be used by later model validation or computation workflows.
