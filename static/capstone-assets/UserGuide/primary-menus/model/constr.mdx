# +Constr

Use **+Constr** when you need to organize objective functions and constraints into reusable sets and collections.

## Where To Find It

1. Select the **Model** primary menu.
2. Click **+Constr** in the secondary button row.

## What It Opens

The **+Constr** dialog has three working areas: **Collections**, **Sets**, and equation assignment controls. You can add collections, add sets, choose whether a set is an **Objective Function** or **Constraint**, attach equations, delete entries, and save the configuration.

## Basic Steps

1. Click **Add Collection** or **Add Set**.
2. Rename the collection or set as needed.
3. For a set, choose **Objective Function** or **Constraint**.
4. Add equation slots and choose equations.
5. Click **Save**.

## Result

Constraint and objective-function groupings are saved for use by model configuration and solution workflows.
