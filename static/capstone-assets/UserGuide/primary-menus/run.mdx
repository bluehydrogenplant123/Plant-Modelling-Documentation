# Run Button

The **Run** button is a primary header action. It does not switch to a secondary menu row; it opens the computation workflow from the header.

## What It Does

Click the linked action name to open the detailed page for the computation panel.

| Action | What It Is For |
| --- | --- |
| **[Computation Panel](./run/computation-panel)** | Start, monitor, or stop computation for the current diagram. |

If the diagram has unsaved changes, **Run** opens a save prompt first. After the prompt, it opens the **Computation Panel**.
