# Run Button

The **Run** button is a primary header action. It does not switch to a secondary menu row; it starts or manages computation from the header.

## What It Does

| Button | What It Is For |
| --- | --- |
| **Run** | Start or stop computation through the computation control. |

Before using **Run**, verify the model and configure solver or algorithm options from **Set Run**.
