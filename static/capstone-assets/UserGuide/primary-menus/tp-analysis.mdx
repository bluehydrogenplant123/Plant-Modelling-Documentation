# TP Analysis Menu

The **TP Analysis** primary menu opens visualization and dashboard tools for time-period analysis.

## Secondary Buttons

Click a button name to open the detailed page for that secondary button.

| Button | What It Is For |
| --- | --- |
| **[Visualization](./tp-analysis/visualization)** | Open the visualization panel backed by Metabase dashboards. |
| **[Dashboard](./tp-analysis/dashboard)** | Open dashboard management tools. |
| **[Edit](./tp-analysis/edit)** | Open the external dashboard editing page in a new browser tab. |
