# TP Analysis Menu

The **TP Analysis** primary menu opens visualization and dashboard tools for time-period analysis.

## Secondary Buttons

| Button | What It Is For |
| --- | --- |
| **Visualization** | Open the visualization panel backed by Metabase dashboards. |
| **Dashboard** | Open dashboard management tools. |
| **Edit** | Open the external dashboard editing page in a new browser tab. |
