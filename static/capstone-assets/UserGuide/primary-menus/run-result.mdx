# Run Result Button

The **Run Result** button is a primary header action for reviewing computation history. It does not switch to a secondary menu row.

## What It Does

Click the linked action name to open the detailed page for run history.

| Action | What It Is For |
| --- | --- |
| **[Run History](./run-result/run-history)** | Open stored computation runs and delete result rows when needed. |

The **Run Result** button can be unavailable when there is no active diagram ID.
