# Simulation

Use **Simulation** when the current network should run in simulation mode.

## Where To Find It

1. Select the **Calc Type** primary menu.
2. Click **Simulation** in the secondary button row.

## What It Opens/Does

If **Simulation** is already selected, clicking it opens **TP Specs**. If another calculation type is selected, clicking **Simulation** opens **Switch Calc Type**. Confirming the switch changes the calculation mode and opens **TP Specs**.

## Basic Steps

1. Open **Calc Type**.
2. Click **Simulation**.
3. If **Switch Calc Type** appears, review the source and target modes.
4. Click **Confirm** to switch to **Simulation**.
5. Review the rows in **TP Specs**.
6. Click **Save** if you changed editable values.

## Result

The network uses **Simulation** as its calculation mode. **TP Specs** shows the specification rows for the selected mode.
