# ParamUpdt

Use **ParamUpdt** when the current network should run in parameter update mode.

## Where To Find It

1. Select the **Calc Type** primary menu.
2. Click **ParamUpdt** in the secondary button row.

## What It Opens/Does

If **ParamUpdt** is already selected, clicking it opens **TP Specs**. If another calculation type is selected, clicking **ParamUpdt** opens **Switch Calc Type**. Confirming the switch changes the calculation mode and opens **TP Specs**.

## Basic Steps

1. Open **Calc Type**.
2. Click **ParamUpdt**.
3. If **Switch Calc Type** appears, review the source and target modes.
4. Click **Confirm** to switch to **ParamUpdt**.
5. Review the rows in **TP Specs**.
6. Edit the required values for parameter update workflows.
7. Click **Save**.

## Result

The network uses **ParamUpdt** as its calculation mode, and the saved **TP Specs** rows are ready for parameter update workflows.
