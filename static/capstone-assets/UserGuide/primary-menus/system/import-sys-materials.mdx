# Import Sys Materials

Use **Import Sys Materials** as a reference for the current system-material import placeholder.

## Where To Find It

1. Open HYPRONET.
2. Select the **System** primary menu.
3. Find **Import Sys Materials** in the secondary button row.

## What It Does

**Import Sys Materials** is currently an inline **System** button. It may be disabled by computation-related rules, and its import handler is a placeholder in the current UI.

It does not currently open a complete system-material import workflow.

## Basic Steps

1. Select the **System** primary menu.
2. Check whether **Import Sys Materials** is enabled.
3. If it is disabled, continue using the existing system materials.
4. If it appears enabled, treat it as a placeholder unless a future version adds an import dialog.

## Result

No system-material import flow is completed from this button in the current implementation.
