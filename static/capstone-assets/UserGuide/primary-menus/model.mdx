# Model Menu

The **Model** primary menu is selected in the annotated screenshot. It controls the main model lifecycle: creating, opening, duplicating, extracting, verifying, importing, and exporting the network.

## Secondary Buttons

Click a button name to open the detailed page for that secondary button.

| Button | What It Is For |
| --- | --- |
| **[New Model](./model/new-model)** | Start a new model diagram. |
| **[Open](./model/open)** | Open an existing network. |
| **[Duplicate Network](./model/duplicate-network)** | Save a copy of the current network. |
| **[Extract Selection](./model/extract-selection)** | Extract selected canvas elements into a separate selection workflow. |
| **[Subnetwork Editor](./model/subnetwork-editor)** | Save or edit a subnetwork from the current model context. |
| **[Import Subnetworks](./model/import-subnetworks)** | Import subnetwork definitions into the current network. |
| **[Equation Writing](./model/equation-writing)** | Open the equation editor for equations and constraints. |
| **[+Constr](./model/constr)** | Open the constraint module. |
| **[SoluAlgoLib](./model/solu-algo-lib)** | Open the solution algorithm library editor. |
| **[Verify Model](./model/verify-model)** | Validate the model before advanced workflows or computation. |
| **[Base TP](./model/base-tp)** | Edit the base time-period duration. |
| **[Specs](./model/specs)** | Open base specification settings. |
| **[Import](./model/import)** | Import diagram data. |
| **[Export](./model/export)** | Export diagram data. |
