# Documentation

Use **Documentation** as a reference for the current in-app documentation placeholder.

## Where To Find It

1. Open HYPRONET.
2. Select the **Help** primary menu.
3. Find **Documentation** in the secondary button row.

## What It Does

**Documentation** is disabled in the current header implementation. It is reserved for future in-app documentation access.

Use the online User Manual when this in-app placeholder is unavailable.

## Basic Steps

1. Select the **Help** primary menu.
2. Confirm **Documentation** is disabled.
3. Open the online User Manual instead.
4. Start from the **Interface Overview** when you need help finding a menu or button.

## Result

No in-app documentation page opens from this button in the current implementation.
