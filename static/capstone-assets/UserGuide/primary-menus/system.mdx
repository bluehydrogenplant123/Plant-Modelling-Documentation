# System Menu

The **System** primary menu contains system-level import placeholders for shared reference data.

## Secondary Buttons

Click a button name to open the detailed page for that secondary button.

| Button | What It Is For |
| --- | --- |
| **[Import Sys Materials](./system/import-sys-materials)** | Placeholder for importing system material data. |
| **[Import Node Definitions](./system/import-node-definitions)** | Placeholder for importing system node definition data. |

These actions are currently inline buttons under the **System** section. They may be disabled by computation-related rules, and their import handlers are placeholders in the current implementation.
