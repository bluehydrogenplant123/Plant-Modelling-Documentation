# System Menu

The **System** primary menu contains system-level import actions for shared reference data.

## Secondary Buttons

| Button | What It Is For |
| --- | --- |
| **Import Sys Materials** | Import system material data. |
| **Import Node Definitions** | Import system node definition data. |

These actions may be disabled while computation-related disable rules are active.
